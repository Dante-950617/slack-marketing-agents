# 프로젝트 팩트시트: 르본 데콘젤 (검증 완료분, 2026-07-04 기준)

르본/데콘젤/허브마스크 관련 리서치 요청이면 이 파일을 먼저 읽고, 여기 있는 검증 완료 사실은 재조사하지 않는다. 단, "검색 현황" 같은 시계열 수치는 시점이 지났으면 최신값으로 갱신한다.

## 제품 팩트시트

- **데콘젤 (데콘게스티브젤)**: 르본코스메틱 × 닥터르노(프랑스, 40년 임상). 아줄렌 기반 진정·쿨링 젤. 240g, 212,100원. 타겟 고민: 홍조·열감·트러블·예민피부. 실사용: 팩붓으로 듬뿍 도포 + 시트 덮는 **관리실형 팩** (얇게 바르는 크림 아님 — 크림 기준 분석 금지)
- **허브마스크**: 시트 타입, 썬번·급성 열감 급속 진정. 여름 수요 대응, 저관여 진입 상품 역할 가능
- **경쟁 정의**: 15만원 이상 고가 에스테틱·더마 제품군
- **유통**: 네이버쇼핑 166개 리스팅, 전부 에스테틱 원장 스마트스토어(오드엘르·여배우스킨·브로우스킨뷰티·에버애프터 등). 전 판매처 212,100원 동일가(가격 질서 세그먼트 1위). 공식몰 lebon.co.kr은 검색 화면에 미노출
- **검색**: '데콘젤'이 '르본'의 5배 (제품이 브랜드를 끄는 구조). 작년 상반기 대비 +84% 성장했으나 26년 5~6월 두 달 연속 전년 하회 — 하락 전환 경고
- **키워드 침식**: '데콘젤' 쇼핑 검색에 저가 미투 혼입 — 셀바이셀 아줄렌 베라 데콘 젤(1L 9.8만), 셀뮬러 데콘 A 젤, 화도 데콘젤 아줄렌 앰플(80ml 2.3만, 카피 문구 유사)
- **후기 자산**: 블로그 누적 1,435건 — 검색 규모 대비 과잉 성취 (시딩 인프라 작동 중의 물증)

## 시장 구조 발견 (검증 완료분만)

- 홍조 검색 성수기는 여름이 아니라 **겨울~환절기(11~3월)**, 2년 연속 재현. 여름은 '수딩젤·쿨링' 언어의 계절 — 같은 제품이 계절마다 다른 언어로 팔려야 함
- 홍조 고민자의 검색 여정: 원인 탐색 > 시술(레이저) > 제품 탐색. 제품 검색자가 가장 적다 → 소재는 제품 자랑이 아니라 **진단형 훅**으로
- 소비자 언어의 본질은 사회적 수치심 ("남들이 무슨 일 있냐고 물어볼 정도", "화장으로 가려도 티가 남")
- 세그먼트 승자 공식: 고민 직결 히어로 SKU + 시술후 수요 편승 + 임상 수치 소구 + 공식몰 검색 장악 + 후기 물량
- '레이저 후' 시장은 1~4만원 매스 가격에서 점유 완료(이지듀 포스트레이저·리쥬란 포스트케어·셀퓨전씨 레이저썬). **프리미엄(15만+) 시술후 '진정'만 공백**
- 온라인 15만원+ 가격 방어 유일 성공 사례: 스킨수티컬즈 (수치 임상 + 히어로 집중 + 공식몰 + 오프라인 연계 서사)
- 마리아갈랑 = 르본의 반면교사: 원장 리셀러 유통 + 공식몰 부재 동일. 차이는 가격 질서 유지 여부 하나

## 확정 액션 백로그 (급한 순)

1. 5~6월 검색 하락 차단 — 여름 '열감·관리실 팩' 워딩 소재 즉시 투입 (수딩젤 검색 7월 피크 편승)
2. 공식몰 검색 화면 진입 — 쇼핑 광고 + 브랜드 검색 등록
3. '데콘젤' 키워드 방어 — 쇼핑 타이틀에 정품·240g·닥터르노 명시, 미투 차별점 상세 최상단
4. 임상의 숫자 번역 — "N주·N명·개선 N%" 문법으로 소재·상세·시딩 일괄 적용
5. 후기 물량전 — 원장 네트워크 시딩, 1차 목표 1,729건(스킨수티컬즈) 추월
